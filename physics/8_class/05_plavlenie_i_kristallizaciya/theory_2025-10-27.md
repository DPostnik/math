## 1. ПОВТОРЕНИЕ (5 минут)

### Что такое количество теплоты?

**Количество теплоты (Q)** - это **энергия**, которую тело получает или отдает при теплопередаче.

**Формулы**:
- **Нагревание/охлаждение**: Q = c · m · Δt
- **Горение**: Q = q · m

**Вопрос для повторения**: Какая формула используется для расчета энергии при нагревании воды?

**Ответ**: Q = c · m · Δt, где c = 4200 Дж/(кг·°C) для воды.

## 2. АГРЕГАТНЫЕ СОСТОЯНИЯ ВЕЩЕСТВА

### Три агрегатных состояния

Любое вещество может находиться в трех состояниях:

1. **Твердое состояние** - частицы расположены в определенном порядке (кристаллическая решетка), сохраняют форму и объем
   - Примеры: лед, железо, соль, сахар

2. **Жидкое состояние** - частицы расположены беспорядочно, вещество сохраняет объем, но не форму
   - Примеры: вода, масло, ртуть

3. **Газообразное состояние** - частицы расположены очень редко, вещество не сохраняет ни форму, ни объем
   - Примеры: водяной пар, воздух, кислород

### Переходы между состояниями

**Важно**: При переходе из одного состояния в другое изменяется расположение частиц вещества, но **само вещество не меняется** (вода остается водой в любом состоянии).

## 3. ПЛАВЛЕНИЕ - ФИЗИЧЕСКОЕ ЯВЛЕНИЕ

### Определение

**Плавление** - это процесс перехода вещества из твердого состояния в жидкое состояние при нагревании.

### Эксперимент 1: Плавление льда

**Оборудование**: Лед, термометр, стакан, источник тепла.

**Ход опыта**:
1. Нагреваем лед в стакане
2. Измеряем температуру через равные промежутки времени
3. Наблюдаем за процессом

**Наблюдения**:
- Пока лед не весь растаял, температура **не меняется** и равна 0°C
- Лед постепенно превращается в воду
- После того, как весь лед растаял, температура воды начинает повышаться

**Вывод**: При плавлении температура **не меняется**, пока вся твердая фаза не превратится в жидкость!

### Эксперимент 2: Плавление свинца (демонстрация)

**Оборудование**: Свинец, тигель, горелка, термометр.

**Ход опыта**:
1. Нагреваем кусок свинца
2. Измеряем температуру

**Наблюдения**:
- Температура свинца повышается до 327°C
- При достижении 327°C температура **перестает расти**
- Свинец начинает плавиться
- Пока весь свинец не расплавился, температура остается 327°C
- После полного плавления температура снова начинает повышаться

**Вывод**: Каждое вещество имеет **свою температуру плавления**, при которой оно плавится.

---

## 4. ТЕМПЕРАТУРА ПЛАВЛЕНИЯ

### Определение

**Температура плавления** - это постоянная температура, при которой твердое вещество плавится (переходит в жидкое состояние).

**Обозначение**: tₚ (t плавления)

**Важно**: 
- Температура плавления **одинакова** для данного вещества
- При плавлении температура **не меняется**, пока не расплавится все твердое вещество
- Для плавления нужно **подводить энергию** (теплоту)

### Таблица температур плавления некоторых веществ

| Вещество | tₚ, °C | Особенность |
|----------|--------|-------------|
| **Водород** | -259 | Очень низкая |
| **Кислород** | -219 | Очень низкая |
| **Азот** | -210 | Очень низкая |
| **Спирт этиловый** | -114 | Низкая |
| **Лед (вода)** | 0 | Знакомая температура |
| **Нафталин** | 80 | Используется против моли |
| **Свинец** | 327 | Металл |
| **Алюминий** | 660 | Металл |
| **Медь** | 1083 | Металл |
| **Железо** | 1539 | Металл |
| **Вольфрам** | 3410 | Самая высокая |

**Важные выводы**:
- **Лед плавится при 0°C** - это важно запомнить!
- У разных веществ разные температуры плавления
- Металлы имеют высокие температуры плавления
- Некоторые вещества (кислород, азот) плавятся при очень низких температурах

## 5. КРИСТАЛЛИЗАЦИЯ (ОТВЕРДЕВАНИЕ)

### Определение

**Кристаллизация** (отвердевание) - это процесс перехода вещества из жидкого состояния в твердое состояние при охлаждении.

### Эксперимент 3: Кристаллизация воды

**Оборудование**: Вода, морозильная камера, термометр.

**Ход опыта**:
1. Охлаждаем воду в морозильной камере
2. Измеряем температуру

**Наблюдения**:
- Температура воды понижается до 0°C
- При достижении 0°C температура **перестает понижаться**
- Вода начинает замерзать (кристаллизоваться)
- Пока вся вода не замерзла, температура остается 0°C
- После полного замерзания температура снова начинает понижаться

**Вывод**: При кристаллизации температура **не меняется**, пока вся жидкость не превратится в твердое вещество!

### Температура кристаллизации

**Температура кристаллизации** - это постоянная температура, при которой жидкость кристаллизуется (переходит в твердое состояние).

**Важно**: 
- Температура кристаллизации **равна** температуре плавления для данного вещества!
- tₖ = tₚ (температура кристаллизации = температура плавления)
- Для воды: tₚ = 0°C, tₖ = 0°C
- При кристаллизации вещество **отдает энергию** (теплоту) окружающей среде

## 6. ГРАФИК ПЛАВЛЕНИЯ И КРИСТАЛЛИЗАЦИИ

### График процесса плавления льда

**Опыт**: Нагреваем лед, начиная с температуры -10°C, и измеряем температуру через равные промежутки времени.

**График зависимости температуры от времени**:

```
t, °C
  ↑
  |                    ┌───────
100|                    |
   |                    |
   |                    |
 10|                    |
   |                    |
  0|───────             |
   |       |            |
-10|───────┴────────────┴─────────→ t, мин
   0       5   10   15   20   25
   
   Область 1: Нагревание льда (от -10°C до 0°C)
   Область 2: Плавление льда (температура 0°C, не меняется!)
   Область 3: Нагревание воды (от 0°C и выше)
```

**Анализ графика**:

**Участок 1 (0-5 минут)**: Нагревание льда
- Температура повышается от -10°C до 0°C
- Лед остается твердым
- Подводится теплота: Q = cльда · m · Δt

**Участок 2 (5-15 минут)**: Плавление льда
- Температура **постоянна** и равна 0°C
- Лед превращается в воду
- Подводится теплота для плавления (об этом на следующем уроке)
- **Горизонтальный участок** - признак плавления!

**Участок 3 (15-25 минут)**: Нагревание воды
- Температура повышается от 0°C
- Вода остается жидкой
- Подводится теплота: Q = cводы · m · Δt

### График процесса кристаллизации воды

**Опыт**: Охлаждаем воду, начиная с температуры 20°C.

**График зависимости температуры от времени**:

```
t, °C
  ↑
  |
 20|───────
   |       |
 10|       |
   |       |
  0|───────┴─────────────
   |                    |
-10|────────────────────┴───────→ t, мин
   0       5   10   15   20   25
   
   Область 1: Охлаждение воды (от 20°C до 0°C)
   Область 2: Кристаллизация воды (температура 0°C, не меняется!)
   Область 3: Охлаждение льда (от 0°C и ниже)
```

**Анализ графика**:

**Участок 1 (0-5 минут)**: Охлаждение воды
- Температура понижается от 20°C до 0°C
- Вода остается жидкой
- Отдается теплота: Q = cводы · m · Δt

**Участок 2 (5-15 минут)**: Кристаллизация воды
- Температура **постоянна** и равна 0°C
- Вода превращается в лед
- Отдается теплота при кристаллизации (об этом на следующем уроке)
- **Горизонтальный участок** - признак кристаллизации!

**Участок 3 (15-25 минут)**: Охлаждение льда
- Температура понижается от 0°C
- Лед остается твердым
- Отдается теплота: Q = cльда · m · Δt

## 7. ОСОБЕННОСТИ ПЛАВЛЕНИЯ И КРИСТАЛЛИЗАЦИИ

### Особенность 1: Температура не меняется

**При плавлении**:
- Температура остается **постоянной** и равна температуре плавления
- Пока не расплавится все твердое вещество, температура не изменится
- Пример: лед плавится при 0°C, и температура не меняется, пока весь лед не растает

**При кристаллизации**:
- Температура остается **постоянной** и равна температуре кристаллизации
- Пока не закристаллизуется вся жидкость, температура не изменится
- Пример: вода замерзает при 0°C, и температура не меняется, пока вся вода не замерзнет

### Особенность 2: Потребность в энергии

**При плавлении**:
- Нужно **подводить энергию** (теплоту) к веществу
- Эта энергия идет на разрушение кристаллической решетки
- Энергия не идет на повышение температуры!

**При кристаллизации**:
- Вещество **отдает энергию** (теплоту) окружающей среде
- Эта энергия выделяется при образовании кристаллической решетки
- Энергия не идет на понижение температуры!

### Особенность 3: Одинаковые температуры

**Температура плавления = Температура кристаллизации**

Для каждого вещества:
- tₚ = tₖ

**Примеры**:
- Вода: tₚ = 0°C, tₖ = 0°C
- Свинец: tₚ = 327°C, tₖ = 327°C
- Алюминий: tₚ = 660°C, tₖ = 660°C

## 8. ПРАКТИЧЕСКИЕ ПРИМЕРЫ ИЗ ЖИЗНИ

### Пример 1: Лед в напитке

**Наблюдение**: Когда кладем лед в стакан с напитком, лед тает, но напиток охлаждается.

**Объяснение**:
- Лед плавится при 0°C
- Для плавления льда нужна энергия
- Эта энергия берется из напитка
- Напиток охлаждается, отдавая энергию льду

### Пример 2: Замерзание воды в трубах

**Наблюдение**: Зимой вода в трубах может замерзнуть и разорвать трубу.

**Объяснение**:
- Вода замерзает при 0°C (кристаллизуется)
- При замерзании вода увеличивается в объеме (это особенность воды)
- Увеличение объема может разорвать трубу

### Пример 3: Плавление металлов

**Наблюдение**: В металлургии металлы плавят в печах при высоких температурах.

**Объяснение**:
- Каждый металл имеет свою температуру плавления
- Железо плавится при 1539°C
- Алюминий плавится при 660°C
- Для плавления нужно подвести много энергии

### Пример 4: Снег на улице

**Наблюдение**: Когда температура воздуха выше 0°C, снег тает.

**Объяснение**:
- Снег (это лед) плавится при 0°C
- Если температура воздуха выше 0°C, снег получает тепло и тает
- Если температура воздуха ниже 0°C, снег не тает

## 9. СВЯЗЬ С ПРЕДЫДУЩИМИ ТЕМАМИ

### Связь с количеством теплоты

**Нагревание твердого тела**:
- Q₁ = cтв · m · Δt (до температуры плавления)

**Плавление**:
- Нужна дополнительная энергия Q₂ (об этом на следующем уроке)

**Нагревание жидкости**:
- Q₃ = cж · m · Δt (после плавления)

**Общая энергия**:
- Qобщая = Q₁ + Q₂ + Q₃

### Связь с температурой

**Важно**: При плавлении и кристаллизации температура **не меняется**, поэтому:
- Нельзя использовать формулу Q = c · m · Δt для плавления!
- Для плавления нужна **другая формула** (изучим на следующем уроке)

## 10. ВАЖНЫЕ ЗАМЕЧАНИЯ

1. **Температура плавления** - это характеристика вещества, она постоянна для данного вещества.

2. **При плавлении** температура не меняется, пока не расплавится все вещество. На графике это **горизонтальный участок**.

3. **При кристаллизации** температура не меняется, пока не закристаллизуется вся жидкость. На графике это **горизонтальный участок**.

4. **Температура плавления = Температура кристаллизации** для каждого вещества.

5. **Для плавления нужна энергия** (теплота), для кристаллизации энергия **выделяется**.

6. **График процесса** - важный инструмент для понимания процессов плавления и кристаллизации.
