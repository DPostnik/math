## ЧТО НУЖНО ВЫУЧИТЬ К СЛЕДУЮЩЕМУ УРОКУ

### Основные понятия (обязательно):

**Агрегатные состояния вещества**:

- **Твердое** - сохраняет форму и объем
- **Жидкое** - сохраняет объем, но не форму
- **Газообразное** - не сохраняет ни форму, ни объем

**Плавление** - переход из твердого в жидкое состояние при нагревании

**Кристаллизация** (отвердевание) - переход из жидкого в твердое состояние при охлаждении

**Температура плавления** - постоянная температура, при которой вещество плавится

**Температура кристаллизации** - постоянная температура, при которой жидкость кристаллизуется

### Важные правила:

1. **Температура плавления = Температура кристаллизации** для каждого вещества

   - Для воды: tₚ = 0°C, tₖ = 0°C

2. **При плавлении** температура **не меняется**, пока не расплавится все вещество

   - На графике это **горизонтальный участок**

3. **При кристаллизации** температура **не меняется**, пока не закристаллизуется вся жидкость

   - На графике это **горизонтальный участок**

4. **Для плавления** нужна энергия (теплота подводится)
5. **При кристаллизации** энергия выделяется (теплота отдается)

### Температуры плавления (выучить наизусть):

| Вещество       | tₚ, °C |
| -------------- | ------ |
| **Лед (вода)** | 0      |
| **Свинец**     | 327    |
| **Алюминий**   | 660    |
| **Медь**       | 1083   |
| **Железо**     | 1539   |

### Графики процессов:

**График плавления**:

- Участок 1: Нагревание твердого вещества (температура повышается)
- Участок 2: **Плавление** (температура постоянна, горизонтальный участок)
- Участок 3: Нагревание жидкости (температура повышается)

**График кристаллизации**:

- Участок 1: Охлаждение жидкости (температура понижается)
- Участок 2: **Кристаллизация** (температура постоянна, горизонтальный участок)
- Участок 3: Охлаждение твердого вещества (температура понижается)

### Формулы для нагревания/охлаждения:

**Нагревание льда**: Q = cльда · m · Δt

- cльда = 2100 Дж/(кг·°C)

**Нагревание воды**: Q = cводы · m · Δt

- cводы = 4200 Дж/(кг·°C)

**Важно**: Эти формулы используются ДО плавления и ПОСЛЕ кристаллизации. Для самого процесса плавления/кристаллизации формула другая (изучим на следующем уроке).

## ИНТЕРЕСНЫЕ ФАКТЫ

- **Вода** - единственное вещество, которое при замерзании расширяется (увеличивается в объеме). Поэтому лед плавает на воде.

- **Температура плавления** зависит от давления. При высоком давлении лед может плавиться при температуре ниже 0°C.

- **Вольфрам** имеет самую высокую температуру плавления среди всех металлов (3410°C). Поэтому его используют для нитей в лампочках.

- **Жидкий азот** закипает при -196°C, то есть его температура кипения ниже температуры плавления воды (0°C).

- При плавлении **химическая структура** вещества не меняется - вода остается водой, только изменяется расположение молекул.

- **Галлий** - металл, который плавится при температуре 29,8°C, то есть он может плавиться в руке человека (температура тела ~36,6°C).
