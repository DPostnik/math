### Задача 1 (базовая - найти Q)

Сколько теплоты нужно передать алюминиевому бруску массой 800 г для нагревания от 25°C до 75°C?

**Подсказка**:

- Используйте формулу Q = c · m · Δt
- Удельная теплоемкость алюминия c = 900 Дж/(кг·°C)
- Не забудьте перевести массу в килограммы

### Задача 3 (средняя - найти температуру)

Воду массой 3 кг нагрели, передав ей 630 кДж теплоты. Начальная температура воды была 20°C. Найдите конечную температуру.

**Подсказка**:

- Сначала найдите изменение температуры Δt из формулы Q = c · m · Δt
- Затем найдите конечную температуру: t₂ = t₁ + Δt

### Задача 4 (средняя - уравнение теплового баланса)

В воду массой 2,5 кг при температуре 18°C опустили железную деталь массой 600 г при температуре 120°C. Определите конечную температуру смеси.

**Подсказка**:

- Вода получает тепло: Q₁ = c₁ · m₁ · (tконеч - t₁)
- Железо отдает тепло: Q₂ = c₂ · m₂ · (tконеч - t₂) < 0
- Уравнение теплового баланса: c₂ · m₂ · (t₂ - tконеч) = c₁ · m₁ · (tконеч - t₁)
- Удельная теплоемкость железа c = 460 Дж/(кг·°C)
- Проверьте: tхолод < tконеч < tгоряч

### Задача 5 (средняя - комбинированная)

В алюминиевую кастрюлю массой 300 г, содержащую 1,2 кг воды при температуре 22°C, налили 0,8 кг кипятка (100°C). Определите конечную температуру воды в кастрюле.

**Подсказка**:

- Учитывайте все три тела: кастрюлю, холодную воду и горячую воду
- Получают тепло: кастрюля и холодная вода
- Отдает тепло: горячая вода
- Уравнение: |Qгорячая вода| = Qкастрюля + Qхолодная вода
- Удельная теплоемкость алюминия c = 900 Дж/(кг·°C)

### Задача 7 (сложная - найти массу одного из тел)

## ЧТО НУЖНО ВЫУЧИТЬ К СЛЕДУЮЩЕМУ УРОКУ

### Основные формулы (обязательно):

1. **Количество теплоты при нагревании/охлаждении**:
   **Q = c · m · Δt**

   где:

   - Q - количество теплоты (тепловая энергия) [Дж]
   - c - удельная теплоемкость [Дж/(кг·°C)]
   - m - масса [кг]
   - Δt = t₂ - t₁ - изменение температуры [°C]

   **Важно**: Количество теплоты - это **энергия**! Измеряется в джоулях (Дж), как и другие виды энергии.

2. **Уравнение теплового баланса**:
   **Qотданное = Qполученное**

   или в развернутом виде:
   **c₁ · m₁ · (t₁горяч - tконеч) = c₂ · m₂ · (tконеч - t₂холод)**

### Алгоритм решения задач (6 шагов):

1. **Анализ условия** - определить, что дано и что найти
2. **Выбор формулы** - Q = cmΔt или уравнение теплового баланса
3. **Перевод в СИ** - граммы → килограммы, кДж → Дж
4. **Составление уравнения** - записать формулу для каждого тела
5. **Решение уравнения** - выразить неизвестную и вычислить
6. **Анализ результата** - проверить единицы и разумность

### Удельные теплоемкости (выучить наизусть):

- **Вода**: c = 4200 Дж/(кг·°C)
- **Железо**: c = 460 Дж/(кг·°C)
- **Медь**: c = 400 Дж/(кг·°C)
- **Алюминий**: c = 900 Дж/(кг·°C)

### Важные правила:

1. Масса всегда в **килограммах** (кг)
2. При нагревании: Q > 0, Δt > 0
3. При охлаждении: Q < 0, Δt < 0
4. При теплообмене: **tхолод < tконеч < tгоряч**
5. При теплообмене учитывайте **все тела**, участвующие в процессе

### Типичные ошибки (избегайте):

- ❌ Не переведены единицы в СИ
- ❌ Неправильный знак при охлаждении
- ❌ Не учтены все тела в теплообмене (например, забыли кастрюлю)
- ❌ Конечная температура вне диапазона начальных температур
- ❌ Ошибки в арифметических вычислениях

## ИНТЕРЕСНЫЕ ФАКТЫ

- Уравнение теплового баланса основано на законе сохранения энергии - энергия не исчезает и не появляется, она только переходит от одного тела к другому.

- В реальной жизни при теплообмене часть тепла всегда уходит в окружающую среду, поэтому уравнение теплового баланса нужно записывать как: Qотданное = Qполученное + Qпотерянное.

- Благодаря большой удельной теплоемкости воды (4200 Дж/(кг·°C)), океаны и моря регулируют климат на Земле - они медленно нагреваются летом и медленно остывают зимой.
