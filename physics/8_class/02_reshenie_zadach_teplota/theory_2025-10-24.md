## 1. ПОВТОРЕНИЕ: Основная формула

### Что такое количество теплоты?

**Количество теплоты (Q)** - это **энергия**, которую тело получает или отдает при теплопередаче.

**Важно понимать**:
- Количество теплоты - это **форма энергии** (тепловая энергия)
- Измеряется в тех же единицах, что и другие виды энергии - в **джоулях (Дж)**
- Энергия может превращаться: механическая → тепловая, химическая → тепловая и т.д.
- При нагревании тело **получает энергию** (Q > 0), при охлаждении - **отдает энергию** (Q < 0)

**Формула для расчета количества теплоты**:
```
Q = c · m · Δt
```

где:
- **Q** - количество теплоты (тепловая энергия) [Дж]
- **c** - удельная теплоемкость [Дж/(кг·°C)]
- **m** - масса [кг]
- **Δt = t₂ - t₁** - изменение температуры [°C]

**Важные правила**:
- При нагревании: Δt > 0, Q > 0 (тело получает энергию)
- При охлаждении: Δt < 0, Q < 0 (тело отдает энергию)
- Масса всегда в килограммах!

## 2. ТИПЫ ЗАДАЧ НА КОЛИЧЕСТВО ТЕПЛОТЫ

### Тип 1: Прямая задача (найти Q)
**Дано**: m, t₁, t₂, c  
**Найти**: Q

**Алгоритм**:
1. Перевести все в СИ (граммы → килограммы)
2. Найти Δt = t₂ - t₁
3. Подставить в формулу: Q = c · m · Δt
4. Вычислить ответ

### Тип 2: Найти массу (m)
**Дано**: Q, t₁, t₂, c  
**Найти**: m

**Алгоритм**:
1. Перевести все в СИ
2. Найти Δt = t₂ - t₁
3. Выразить из формулы: m = Q / (c · Δt)
4. Вычислить ответ

### Тип 3: Найти температуру (t₂)
**Дано**: Q, m, t₁, c  
**Найти**: t₂

**Алгоритм**:
1. Перевести все в СИ
2. Выразить Δt: Δt = Q / (c · m)
3. Найти t₂: t₂ = t₁ + Δt
4. Вычислить ответ

### Тип 4: Найти удельную теплоемкость (c)
**Дано**: Q, m, t₁, t₂  
**Найти**: c

**Алгоритм**:
1. Перевести все в СИ
2. Найти Δt = t₂ - t₁
3. Выразить: c = Q / (m · Δt)
4. Вычислить ответ

## 3. УРАВНЕНИЕ ТЕПЛОВОГО БАЛАНСА

### Физическое явление: Теплообмен

Когда горячее тело соприкасается с холодным, происходит теплообмен:
- Горячее тело **отдает энергию** в виде теплоты (Q₁ < 0)
- Холодное тело **получает энергию** в виде теплоты (Q₂ > 0)
- Процесс продолжается до установления равновесия

**Важно**: Теплообмен происходит до тех пор, пока температуры не станут равными!

### Уравнение теплового баланса

**Формулировка**: Количество теплоты (энергия), отданное горячим телом, равно количеству теплоты (энергии), полученному холодным телом (если тепло не уходит в окружающую среду).

**Физический смысл**: Это проявление **закона сохранения энергии** - энергия не исчезает и не появляется, она только переходит от одного тела к другому при теплообмене.

**Математическая запись**:
```
Qотданное = Qполученное
```

или
```
|Q₁| = Q₂
```

где:
- Q₁ - тепло, отданное горячим телом (Q₁ < 0)
- Q₂ - тепло, полученное холодным телом (Q₂ > 0)

### Формула для теплообмена

Если горячее тело (температура tгоряч, масса m₁, теплоемкость c₁) отдает тепло холодному телу (температура tхолод, масса m₂, теплоемкость c₂), и они достигают температуры tконеч:

**Для горячего тела** (отдает тепло):
```
Q₁ = c₁ · m₁ · (tконеч - tгоряч) < 0
```

**Для холодного тела** (получает тепло):
```
Q₂ = c₂ · m₂ · (tконеч - tхолод) > 0
```

**Уравнение теплового баланса**:
```
c₁ · m₁ · (tгоряч - tконеч) = c₂ · m₂ · (tконеч - tхолод)
```

(берем модули, поэтому меняем порядок вычитания)

## 5. ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 1: Простая задача на нагревание

**Задача**: Сколько теплоты нужно передать алюминиевому бруску массой 500 г для нагревания от 20°C до 80°C?

**Дано**:
- m = 500 г = 0,5 кг
- t₁ = 20°C
- t₂ = 80°C
- c = 900 Дж/(кг·°C) (алюминий)

**Найти**: Q

**Решение**:
1. Δt = t₂ - t₁ = 80°C - 20°C = 60°C
2. Q = c · m · Δt = 900 Дж/(кг·°C) · 0,5 кг · 60°C = 27 000 Дж

**Ответ**: Q = 27 000 Дж = 27 кДж

---

### Пример 2: Задача на охлаждение

**Задача**: Какое количество теплоты отдаст железная деталь массой 2 кг при охлаждении от 300°C до 50°C?

**Дано**:
- m = 2 кг
- t₁ = 300°C
- t₂ = 50°C
- c = 460 Дж/(кг·°C) (железо)

**Найти**: Q (количество отданной теплоты)

**Решение**:
1. Δt = t₂ - t₁ = 50°C - 300°C = -250°C
2. Q = c · m · Δt = 460 Дж/(кг·°C) · 2 кг · (-250°C) = -230 000 Дж

Знак минус показывает, что тепло отдается. По модулю:

**Ответ**: |Q| = 230 000 Дж = 230 кДж (деталь отдала 230 кДж теплоты)

---

### Пример 3: Задача на нахождение массы

**Задача**: Какая масса воды получила 840 кДж теплоты при нагревании от 10°C до 60°C?

**Дано**:
- Q = 840 кДж = 840 000 Дж
- t₁ = 10°C
- t₂ = 60°C
- c = 4200 Дж/(кг·°C) (вода)

**Найти**: m

**Решение**:
1. Δt = t₂ - t₁ = 60°C - 10°C = 50°C
2. Из формулы Q = c · m · Δt выразим m:
   m = Q / (c · Δt)
3. m = 840 000 Дж / (4200 Дж/(кг·°C) · 50°C) = 840 000 / 210 000 = 4 кг

**Ответ**: m = 4 кг

### Пример 4: Уравнение теплового баланса (простая)

**Задача**: В воду массой 2 кг при температуре 20°C опустили железную гирю массой 500 г при температуре 100°C. Определите конечную температуру смеси. (Теплообмен только между водой и гирей)

**Дано**:
- Вода: m₁ = 2 кг, t₁ = 20°C, c₁ = 4200 Дж/(кг·°C)
- Железо: m₂ = 500 г = 0,5 кг, t₂ = 100°C, c₂ = 460 Дж/(кг·°C)

**Найти**: tконеч (конечная температура)

**Решение**:
1. Вода получает тепло: Q₁ = c₁ · m₁ · (tконеч - t₁) > 0
2. Железо отдает тепло: Q₂ = c₂ · m₂ · (tконеч - t₂) < 0

Уравнение теплового баланса: |Q₂| = Q₁
```
c₂ · m₂ · (t₂ - tконеч) = c₁ · m₁ · (tконеч - t₁)
```

3. Подставим значения:
```
460 · 0,5 · (100 - tконеч) = 4200 · 2 · (tконеч - 20)
230 · (100 - tконеч) = 8400 · (tконеч - 20)
23000 - 230tконеч = 8400tконеч - 168000
23000 + 168000 = 8400tконеч + 230tконеч
191000 = 8630tконеч
tконеч = 191000 / 8630 ≈ 22,1°C
```

**Ответ**: tконеч ≈ 22,1°C

**Проверка**: 20°C < 22,1°C < 100°C ✓ (температура между начальными)

### Пример 5: Комбинированная задача (несколько тел)

**Задача**: В алюминиевую кастрюлю массой 400 г, содержащую 1,5 кг воды при температуре 20°C, налили 0,5 кг кипятка (100°C). Определите конечную температуру воды в кастрюле. (Теплообмен только между кастрюлей, холодной и горячей водой)

**Дано**:
- Кастрюля (алюминий): m₁ = 400 г = 0,4 кг, t₁ = 20°C, c₁ = 900 Дж/(кг·°C)
- Холодная вода: m₂ = 1,5 кг, t₂ = 20°C, c₂ = 4200 Дж/(кг·°C)
- Горячая вода: m₃ = 0,5 кг, t₃ = 100°C, c₃ = 4200 Дж/(кг·°C)

**Найти**: tконеч

**Решение**:
1. Получают тепло (нагреваются до tконеч):
   - Кастрюля: Q₁ = c₁ · m₁ · (tконеч - t₁)
   - Холодная вода: Q₂ = c₂ · m₂ · (tконеч - t₂)

2. Отдает тепло (охлаждается до tконеч):
   - Горячая вода: Q₃ = c₃ · m₃ · (tконеч - t₃) < 0

3. Уравнение теплового баланса: |Q₃| = Q₁ + Q₂
```
c₃ · m₃ · (t₃ - tконеч) = c₁ · m₁ · (tконеч - t₁) + c₂ · m₂ · (tконеч - t₂)
```

4. Подставим значения:
```
4200 · 0,5 · (100 - tконеч) = 900 · 0,4 · (tконеч - 20) + 4200 · 1,5 · (tконеч - 20)
2100 · (100 - tконеч) = 360 · (tконеч - 20) + 6300 · (tконеч - 20)
210000 - 2100tконеч = (360 + 6300) · (tконеч - 20)
210000 - 2100tконеч = 6660 · (tконеч - 20)
210000 - 2100tконеч = 6660tконеч - 133200
210000 + 133200 = 6660tконеч + 2100tконеч
343200 = 8760tконеч
tконеч = 343200 / 8760 ≈ 39,2°C
```

**Ответ**: tконеч ≈ 39,2°C

## 6. ТИПИЧНЫЕ ОШИБКИ И КАК ИХ ИЗБЕЖАТЬ

### Ошибка 1: Не переведены единицы в СИ
❌ **Неправильно**: m = 500 г, подставляем в формулу
✅ **Правильно**: m = 500 г = 0,5 кг

### Ошибка 2: Неправильный знак при охлаждении
❌ **Неправильно**: Δt = t₁ - t₂ (при охлаждении)
✅ **Правильно**: Δt = t₂ - t₁ (всегда конечная минус начальная)

### Ошибка 3: Не учтены все тела в теплообмене
❌ **Неправильно**: Учитываем только воду, забываем кастрюлю
✅ **Правильно**: Учитываем все тела, участвующие в теплообмене

### Ошибка 4: Неправильное уравнение теплового баланса
❌ **Неправильно**: Q₁ = Q₂ (без учета знаков)
✅ **Правильно**: |Qотданное| = Qполученное

### Ошибка 5: Конечная температура вне диапазона
❌ **Неправильно**: tконеч = 150°C, когда tгоряч = 100°C
✅ **Правильно**: tхолод < tконеч < tгоряч

## 7. ВАЖНЫЕ ЗАМЕЧАНИЯ

1. **Количество теплоты - это энергия**: При теплообмене энергия не исчезает и не появляется - она переходит от горячего тела к холодному. Это закон сохранения энергии.

2. **Конечная температура** всегда находится между начальными температурами тел.

3. **Если тепло уходит в окружающую среду**, уравнение теплового баланса становится: Qотданное = Qполученное + Qпотерянное (энергия распределяется между телами и окружающей средой)

4. **При расчетах** всегда проверяйте разумность результата!

5. **Единицы измерения** должны быть согласованы (все в СИ). Количество теплоты измеряется в джоулях (Дж), как и любая энергия.
