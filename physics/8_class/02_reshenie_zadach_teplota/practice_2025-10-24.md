## ПОВТОРЕНИЕ (10 минут)

**Напоминание**: Количество теплоты (Q) - это **энергия**, которую тело получает или отдает при теплопередаче. Измеряется в джоулях (Дж).

### Задача 1 (легкая)
Найдите количество теплоты (энергию), необходимое для нагревания воды массой 3 кг от 15°C до 85°C.

**Дано**:
- m = 3 кг
- t₁ = 15°C
- t₂ = 85°C
- c = 4200 Дж/(кг·°C) (вода)

**Решение**: 
1. Δt = t₂ - t₁ = 85°C - 15°C = 70°C
2. Q = c · m · Δt = 4200 Дж/(кг·°C) · 3 кг · 70°C = 882 000 Дж

**Ответ**: Q = 882 000 Дж = 882 кДж

### Задача 2 (легкая)
Какое количество теплоты отдаст медная деталь массой 1,5 кг при охлаждении от 200°C до 50°C?

**Дано**:
- m = 1,5 кг
- t₁ = 200°C
- t₂ = 50°C
- c = 400 Дж/(кг·°C) (медь)

**Решение**:
1. Δt = t₂ - t₁ = 50°C - 200°C = -150°C
2. Q = c · m · Δt = 400 Дж/(кг·°C) · 1,5 кг · (-150°C) = -90 000 Дж

**Ответ**: |Q| = 90 000 Дж = 90 кДж (деталь отдала 90 кДж теплоты)

## НОВАЯ ТЕМА: Разнообразные задачи (20-25 минут)

### Задача 3 (средняя - найти массу)
Какая масса воды получила 630 кДж теплоты при нагревании от 20°C до 50°C?

**Дано**:
- Q = 630 кДж = 630 000 Дж
- t₁ = 20°C
- t₂ = 50°C
- c = 4200 Дж/(кг·°C) (вода)

**Найти**: m

**Решение**:
1. Δt = t₂ - t₁ = 50°C - 20°C = 30°C
2. Из формулы Q = c · m · Δt выразим m:
   m = Q / (c · Δt)
3. m = 630 000 Дж / (4200 Дж/(кг·°C) · 30°C) = 630 000 / 126 000 = 5 кг

**Ответ**: m = 5 кг

### Задача 4 (средняя - найти конечную температуру)
Воду массой 2 кг нагрели, передав ей 420 кДж теплоты. Начальная температура воды была 25°C. Найдите конечную температуру.

**Дано**:
- m = 2 кг
- Q = 420 кДж = 420 000 Дж
- t₁ = 25°C
- c = 4200 Дж/(кг·°C) (вода)

**Найти**: t₂

**Решение**:
1. Из формулы Q = c · m · Δt найдем Δt:
   Δt = Q / (c · m) = 420 000 Дж / (4200 Дж/(кг·°C) · 2 кг) = 420 000 / 8400 = 50°C
2. t₂ = t₁ + Δt = 25°C + 50°C = 75°C

**Ответ**: t₂ = 75°C

### Задача 5 (средняя - найти удельную теплоемкость)
Тело массой 4 кг получило 920 кДж теплоты и нагрелось от 10°C до 60°C. Определите удельную теплоемкость вещества.

**Дано**:
- m = 4 кг
- Q = 920 кДж = 920 000 Дж
- t₁ = 10°C
- t₂ = 60°C

**Найти**: c

**Решение**:
1. Δt = t₂ - t₁ = 60°C - 10°C = 50°C
2. Из формулы Q = c · m · Δt выразим c:
   c = Q / (m · Δt)
3. c = 920 000 Дж / (4 кг · 50°C) = 920 000 / 200 = 4600 Дж/(кг·°C)

**Ответ**: c = 4600 Дж/(кг·°C)

### Задача 6 (средняя - уравнение теплового баланса)
В воду массой 3 кг при температуре 15°C опустили железную деталь массой 800 г при температуре 150°C. Определите конечную температуру смеси. (Теплообмен только между водой и железом)

**Дано**:
- Вода: m₁ = 3 кг, t₁ = 15°C, c₁ = 4200 Дж/(кг·°C)
- Железо: m₂ = 800 г = 0,8 кг, t₂ = 150°C, c₂ = 460 Дж/(кг·°C)

**Найти**: tконеч

**Решение**:
1. Вода получает тепло: Q₁ = c₁ · m₁ · (tконеч - t₁)
2. Железо отдает тепло: Q₂ = c₂ · m₂ · (tконеч - t₂) < 0

3. Уравнение теплового баланса: |Q₂| = Q₁
```
c₂ · m₂ · (t₂ - tконеч) = c₁ · m₁ · (tконеч - t₁)
```

4. Подставим значения:
```
460 · 0,8 · (150 - tконеч) = 4200 · 3 · (tконеч - 15)
368 · (150 - tконеч) = 12600 · (tконеч - 15)
55200 - 368tконеч = 12600tконеч - 189000
55200 + 189000 = 12600tконеч + 368tконеч
244200 = 12968tконеч
tконеч = 244200 / 12968 ≈ 18,8°C
```

**Ответ**: tконеч ≈ 18,8°C

**Проверка**: 15°C < 18,8°C < 150°C ✓

### Задача 7 (сложная - комбинированная)
В алюминиевую кастрюлю массой 500 г, содержащую 2 кг воды при температуре 18°C, налили 1 кг кипятка (100°C). Определите конечную температуру воды в кастрюле.

**Дано**:
- Кастрюля (алюминий): m₁ = 500 г = 0,5 кг, t₁ = 18°C, c₁ = 900 Дж/(кг·°C)
- Холодная вода: m₂ = 2 кг, t₂ = 18°C, c₂ = 4200 Дж/(кг·°C)
- Горячая вода: m₃ = 1 кг, t₃ = 100°C, c₃ = 4200 Дж/(кг·°C)

**Найти**: tконеч

**Решение**:
1. Получают тепло (нагреваются до tконеч):
   - Кастрюля: Q₁ = c₁ · m₁ · (tконеч - t₁)
   - Холодная вода: Q₂ = c₂ · m₂ · (tконеч - t₂)

2. Отдает тепло (охлаждается до tконеч):
   - Горячая вода: Q₃ = c₃ · m₃ · (tконеч - t₃) < 0

3. Уравнение теплового баланса: |Q₃| = Q₁ + Q₂
```
c₃ · m₃ · (t₃ - tконеч) = c₁ · m₁ · (tконеч - t₁) + c₂ · m₂ · (tконеч - t₂)
```

4. Подставим значения:
```
4200 · 1 · (100 - tконеч) = 900 · 0,5 · (tконеч - 18) + 4200 · 2 · (tконеч - 18)
4200 · (100 - tконеч) = 450 · (tконеч - 18) + 8400 · (tконеч - 18)
420000 - 4200tконеч = (450 + 8400) · (tконеч - 18)
420000 - 4200tконеч = 8850 · (tконеч - 18)
420000 - 4200tконеч = 8850tконеч - 159300
420000 + 159300 = 8850tконеч + 4200tконеч
579300 = 13050tконеч
tконеч = 579300 / 13050 ≈ 44,4°C
```

**Ответ**: tконеч ≈ 44,4°C

### Задача 8 (сложная - с переводом единиц)
В медную кастрюлю массой 600 г при температуре 20°C налили 1,5 л воды при температуре 80°C. Определите конечную температуру воды. Плотность воды 1000 кг/м³.

**Дано**:
- Кастрюля (медь): m₁ = 600 г = 0,6 кг, t₁ = 20°C, c₁ = 400 Дж/(кг·°C)
- Вода: V = 1,5 л = 0,0015 м³, t₂ = 80°C, c₂ = 4200 Дж/(кг·°C), ρ = 1000 кг/м³

**Найти**: tконеч

**Решение**:
1. Найдем массу воды: m₂ = ρ · V = 1000 кг/м³ · 0,0015 м³ = 1,5 кг

2. Кастрюля получает тепло (нагревается): Q₁ = c₁ · m₁ · (tконеч - t₁)
3. Вода отдает тепло (охлаждается): Q₂ = c₂ · m₂ · (tконеч - t₂) < 0

4. Уравнение теплового баланса: |Q₂| = Q₁
```
c₂ · m₂ · (t₂ - tконеч) = c₁ · m₁ · (tконеч - t₁)
```

5. Подставим значения:
```
4200 · 1,5 · (80 - tконеч) = 400 · 0,6 · (tконеч - 20)
6300 · (80 - tконеч) = 240 · (tконеч - 20)
504000 - 6300tконеч = 240tконеч - 4800
504000 + 4800 = 240tконеч + 6300tконеч
508800 = 6540tконеч
tконеч = 508800 / 6540 ≈ 77,8°C
```

**Ответ**: tконеч ≈ 77,8°C

### Задача 9
В сосуд с водой при температуре 20°C опустили два тела:
- Алюминиевый брусок массой 500 г при температуре 200°C
- Железную деталь массой 300 г при температуре 150°C

Масса воды 2 кг. Определите конечную температуру, если все тела достигли теплового равновесия.

**Решение**:

   - Вода: Q₁ = c₁ · m₁ · (tконеч - t₁)

```
c₂ · m₂ · (t₂ - tконеч) + c₃ · m₃ · (t₃ - tконеч) = c₁ · m₁ · (tконеч - t₁)
```

### Задача 10
Сколько кипятка (100°C) нужно добавить к 5 кг воды при температуре 10°C, чтобы получить воду температурой 40°C?

**Дано**:
- Холодная вода: m₁ = 5 кг, t₁ = 10°C, c = 4200 Дж/(кг·°C)
- Горячая вода: t₂ = 100°C, c = 4200 Дж/(кг·°C)
- tконеч = 40°C

**Найти**: m₂ (масса кипятка)

**Решение**:
1. Холодная вода получает тепло: Q₁ = c · m₁ · (tконеч - t₁)
2. Горячая вода отдает тепло: Q₂ = c · m₂ · (tконеч - t₂) < 0

3. Уравнение теплового баланса: |Q₂| = Q₁
```
c · m₂ · (t₂ - tконеч) = c · m₁ · (tконеч - t₁)
```

4. Сократим c (одинаковое):
```
m₂ · (t₂ - tконеч) = m₁ · (tконеч - t₁)
m₂ · (100 - 40) = 5 · (40 - 10)
m₂ · 60 = 5 · 30
m₂ = 150 / 60 = 2,5 кг
```

**Ответ**: m₂ = 2,5 кг
