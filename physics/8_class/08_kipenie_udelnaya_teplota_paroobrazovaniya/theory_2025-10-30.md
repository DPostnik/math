## 1. ПОВТОРЕНИЕ (5 минут)

### Что такое испарение?

**Испарение** - это процесс перехода вещества из жидкого состояния в газообразное (пар) при любой температуре, происходящий с поверхности жидкости.

**Конденсация** - это процесс перехода вещества из газообразного состояния (пара) в жидкое состояние.

**Факторы скорости испарения**:
1. Температура (↑ температура → ↑ скорость)
2. Площадь поверхности (↑ площадь → ↑ скорость)
3. Скорость движения воздуха (↑ ветер → ↑ скорость)
4. Влажность воздуха (↑ влажность → ↓ скорость)
5. Род вещества

**Вопрос для повторения**: При какой температуре происходит испарение?

**Ответ**: Испарение происходит при любой температуре, не только при кипении!

---

## 2. КИПЕНИЕ - ФИЗИЧЕСКОЕ ЯВЛЕНИЕ

### Наблюдение из жизни

**Примеры кипения**:
- Вода в чайнике закипает при 100°C
- Молоко кипит при температуре чуть выше 100°C
- Спирт кипит при 78°C
- Жидкий азот кипит при -196°C

**Вывод**: Разные жидкости кипят при разных температурах!

### Определение

**Кипение** - это процесс интенсивного парообразования, происходящий во всем объеме жидкости при определенной температуре (температуре кипения).

**Важно понимать**:
- Кипение происходит **во всем объеме** жидкости (не только с поверхности!)
- Кипение происходит **при определенной температуре** (температуре кипения)
- При кипении образуются **пузырьки пара** внутри жидкости
- Температура жидкости при кипении **не меняется**, пока вся жидкость не выкипит

---

## 3. ОТЛИЧИЕ КИПЕНИЯ ОТ ИСПАРЕНИЯ

### Сравнение процессов

| Характеристика | Испарение | Кипение |
|----------------|-----------|---------|
| **Где происходит** | С поверхности | Во всем объеме |
| **Температура** | При любой температуре | Только при температуре кипения |
| **Скорость** | Медленное | Быстрое, интенсивное |
| **Пузырьки** | Нет | Да, пузырьки пара |
| **Температура меняется?** | Да, понижается | Нет, постоянна |

**Важно**: 
- **Испарение** - медленный процесс с поверхности при любой температуре
- **Кипение** - быстрый процесс во всем объеме при определенной температуре

## 4. ТЕМПЕРАТУРА КИПЕНИЯ

### Определение

**Температура кипения** - это постоянная температура, при которой жидкость кипит.

**Обозначение**: tₖ (t кипения)

**Важно**: 
- Температура кипения **одинакова** для данного вещества при нормальном атмосферном давлении
- При кипении температура **не меняется**, пока не выкипит вся жидкость
- Для кипения нужно **подводить энергию** (теплоту)

### Таблица температур кипения некоторых веществ

| Вещество | tₖ, °C | Особенность |
|----------|--------|-------------|
| **Жидкий азот** | -196 | Очень низкая |
| **Жидкий кислород** | -183 | Очень низкая |
| **Спирт этиловый** | 78 | Низкая |
| **Вода** | 100 | Знакомая температура |
| **Масло растительное** | ~200 | Высокая |
| **Ртуть** | 357 | Высокая |
| **Свинец** | 1740 | Очень высокая |

**Важные выводы**:
- **Вода кипит при 100°C** - это важно запомнить!
- У разных веществ разные температуры кипения
- Температура кипения зависит от атмосферного давления (в горах вода кипит при более низкой температуре!)

## 5. ПРОЦЕСС КИПЕНИЯ

### Как происходит кипение?

**Этапы процесса кипения**:

1. **Нагревание жидкости**
   - Температура повышается
   - Появляются маленькие пузырьки воздуха на дне и стенках сосуда

2. **Образование пузырьков пара**
   - При достижении температуры кипения внутри жидкости образуются пузырьки пара
   - Пузырьки растут и поднимаются вверх

3. **Кипение**
   - Пузырьки пара лопаются на поверхности
   - Пар выходит в атмосферу
   - Температура остается постоянной (равна температуре кипения)

4. **Продолжение кипения**
   - Пока вся жидкость не выкипит, температура не меняется
   - После выкипания температура пара начинает повышаться

### Эксперимент: Наблюдение кипения воды

**Оборудование**: Колба с водой, горелка, термометр.

**Ход опыта**:
1. Нагреваем воду в колбе
2. Измеряем температуру через равные промежутки времени
3. Наблюдаем за процессом

**Наблюдения**:
- Температура воды повышается до 100°C
- При достижении 100°C появляются пузырьки пара
- Температура **перестает расти** и остается 100°C
- Пузырьки пара лопаются на поверхности
- Пока вся вода не выкипит, температура остается 100°C

**Вывод**: При кипении температура **не меняется**, пока не выкипит вся жидкость!

## 6. УДЕЛЬНАЯ ТЕПЛОТА ПАРООБРАЗОВАНИЯ

### Наблюдение: Энергия для парообразования

**Опыт**: Нагреем воду от 0°C до 100°C, затем превратим в пар.

**Наблюдения**:
- Для нагревания 1 кг воды от 0°C до 100°C нужно ~420 кДж теплоты
- Для превращения 1 кг воды при 100°C в пар нужно **2260 кДж** теплоты!
- Это **в 5,4 раза больше**, чем для нагревания на 100°C!

**Вывод**: Для парообразования нужна **очень большая энергия**, даже если температура не меняется!

### Определение

**Удельная теплота парообразования (L)** - это физическая величина, которая показывает, какое количество теплоты нужно передать жидкости массой 1 кг, чтобы полностью превратить ее в пар при температуре кипения.

**Обозначение**: L (эль)

**Единица измерения**: [L] = Дж/кг или кДж/кг

**Важно понимать**:
- Удельная теплота парообразования показывает, сколько энергии нужно для превращения 1 кг жидкости в пар
- Эта энергия идет на преодоление сил притяжения между молекулами
- Температура при этом не меняется!

### Формула удельной теплоты парообразования

```
L = Q / m
```

где:
- **L** - удельная теплота парообразования [Дж/кг] или [кДж/кг]
- **Q** - количество теплоты, необходимое для парообразования [Дж] или [кДж]
- **m** - масса превратившейся в пар жидкости [кг]

---

## 7. ОСНОВНАЯ ФОРМУЛА ДЛЯ РАСЧЕТА КОЛИЧЕСТВА ТЕПЛОТЫ ПРИ ПАРООБРАЗОВАНИИ

Перепишем формулу для нахождения количества теплоты:

### Формула парообразования:

```
Q = L · m
```

где:
- **Q** - количество теплоты, необходимое для парообразования [Дж] или [кДж]
- **L** - удельная теплота парообразования [Дж/кг] или [кДж/кг]
- **m** - масса превратившейся в пар жидкости [кг]

**Важно**:
- При парообразовании энергия **подводится** к жидкости, поэтому Q > 0
- Формула показывает, сколько энергии нужно для превращения жидкости в пар
- Температура во время парообразования **не меняется**!

---

## 8. УДЕЛЬНАЯ ТЕПЛОТА КОНДЕНСАЦИИ

### Определение

**Удельная теплота конденсации** - это физическая величина, которая показывает, какое количество теплоты выделяется при конденсации (превращении в жидкость) пара массой 1 кг при температуре конденсации.

**Важно**: 
- **Удельная теплота конденсации = Удельная теплота парообразования** для данного вещества!
- Lₖ = Lₚ (L конденсации = L парообразования)

**Формула конденсации**:

```
Q = L · m
```

где:
- **Q** - количество теплоты, выделившееся при конденсации [Дж] или [кДж]
- **L** - удельная теплота конденсации (равна удельной теплоте парообразования) [Дж/кг] или [кДж/кг]
- **m** - масса сконденсировавшегося пара [кг]

**Важно**:
- При конденсации энергия **выделяется**, поэтому Q > 0 (энергия отдается окружающей среде)
- Формула такая же, как для парообразования!

## 10. ПРАКТИЧЕСКИЕ ПРИМЕРЫ ИЗ ЖИЗНИ

### Пример 1: Кипячение воды в чайнике

**Наблюдение**: Вода в чайнике закипает при 100°C и продолжает кипеть.

**Расчет**: Сколько энергии нужно для превращения 2 кг воды при 100°C в пар?

**Решение**:
- m = 2 кг
- L = 2260 кДж/кг = 2 260 000 Дж/кг (вода)
- Q = L · m = 2 260 000 Дж/кг · 2 кг = 4 520 000 Дж = 4520 кДж

**Ответ**: Нужно 4520 кДж энергии. Это очень много! Поэтому чайник долго кипит.

### Пример 2: Конденсация пара

**Наблюдение**: Пар от кипящей воды конденсируется на холодной крышке кастрюли.

**Расчет**: Сколько энергии выделится при конденсации 0,5 кг пара?

**Решение**:
- m = 0,5 кг
- L = 2260 кДж/кг = 2 260 000 Дж/кг (удельная теплота конденсации воды)
- Q = L · m = 2 260 000 Дж/кг · 0,5 кг = 1 130 000 Дж = 1130 кДж

**Ответ**: Выделится 1130 кДж энергии. Эта энергия нагревает крышку кастрюли.

### Пример 3: Паровая баня

**Наблюдение**: В паровой бане используется пар для нагревания.

**Объяснение**: 
- Пар имеет очень большую энергию (удельная теплота парообразования воды 2260 кДж/кг)
- При конденсации эта энергия выделяется
- Поэтому пар может сильно нагреть предметы

## 11. ПОЛНЫЙ ПРОЦЕСС: НАГРЕВАНИЕ + ПАРООБРАЗОВАНИЕ

### Этапы процесса нагревания жидкости до пара

Когда мы нагреваем жидкость от низкой температуры до пара, процесс состоит из **двух этапов**:

**Этап 1: Нагревание жидкости**
- От начальной температуры t₁ до температуры кипения tₖ
- Формула: Q₁ = cж · m · Δt₁, где Δt₁ = tₖ - t₁

**Этап 2: Парообразование (кипение)**
- При температуре кипения tₖ (температура не меняется!)
- Формула: Q₂ = L · m

**Общее количество теплоты**:
```
Qобщая = Q₁ + Q₂
```

### Пример: Нагревание воды до пара

**Задача**: Вода массой 1 кг при температуре 20°C нагрета до пара при температуре 100°C. Найдите общее количество теплоты.

**Дано**:
- m = 1 кг
- t₁ = 20°C (начальная температура воды)
- tₖ = 100°C (температура кипения)
- cводы = 4200 Дж/(кг·°C)
- L = 2 260 000 Дж/кг (удельная теплота парообразования воды)

**Решение**:

**Этап 1**: Нагревание воды от 20°C до 100°C
- Δt₁ = 100°C - 20°C = 80°C
- Q₁ = cводы · m · Δt₁ = 4200 Дж/(кг·°C) · 1 кг · 80°C = 336 000 Дж = 336 кДж

**Этап 2**: Парообразование воды при 100°C
- Q₂ = L · m = 2 260 000 Дж/кг · 1 кг = 2 260 000 Дж = 2260 кДж

**Общее количество теплоты**:
- Qобщая = Q₁ + Q₂ = 336 кДж + 2260 кДж = 2596 кДж = 2,596 МДж

**Ответ**: Qобщая = 2596 кДж = 2,596 МДж

**Важно**: Больше всего энергии (2260 кДж) ушло на парообразование!

## 12. ГРАФИК ПОЛНОГО ПРОЦЕССА С УЧЕТОМ КИПЕНИЯ

**График зависимости температуры от времени** для процесса нагревания воды от 20°C до пара при 100°C:

```
t, °C
  ↑
  |
100|───────┴─────────────
   |       |
   |       |
   |       |
 50|       |
   |       |
 20|───────┴─────────────→ t, мин
   0       5   10   15   20
   
   Участок 1: Нагревание воды (20°C → 100°C)
             Q₁ = cводы · m · Δt₁
   Участок 2: Кипение (парообразование) (100°C, постоянная температура)
             Q₂ = L · m
```

**Анализ графика**:

**Участок 1 (0-5 минут)**: Нагревание воды
- Температура повышается от 20°C до 100°C
- Подводится теплота: Q₁ = cводы · m · Δt₁

**Участок 2 (5-20 минут)**: Кипение (парообразование)
- Температура **постоянна** и равна 100°C
- Подводится теплота для парообразования: Q₂ = L · m
- **Горизонтальный участок** - признак кипения!

**Общая теплота**: Qобщая = Q₁ + Q₂

## 13. ПОШАГОВОЕ РЕШЕНИЕ ЗАДАЧ

### Алгоритм решения задач на парообразование и конденсацию:

**Шаг 1: Анализ условия**
- Определить вещество
- Выписать известные величины (m, L, Q или t)
- Определить, что нужно найти (Q, L, m или полный процесс)

**Шаг 2: Выбор формулы**
- Для парообразования: Q = L · m
- Для конденсации: Q = L · m (та же формула!)
- Для полного процесса: Qобщая = Q₁ + Q₂

**Шаг 3: Перевод в систему СИ**
- Масса: граммы → килограммы (г → кг, разделить на 1000)
- Количество теплоты: кДж → Дж (× 1000), МДж → Дж (× 1 000 000)
- Удельная теплота парообразования: кДж/кг → Дж/кг (× 1000)

**Шаг 4: Подстановка и вычисления**
- Подставить все значения в формулу
- Выполнить вычисления
- Проверить единицы измерения

**Шаг 5: Анализ результата**
- Проверить правильность единиц измерения
- Проверить разумность результата
- Записать ответ с единицами

## 14. ПРИМЕРЫ РЕШЕНИЯ ЗАДАЧ

### Пример 1: Найти количество теплоты при парообразовании

**Задача**: Сколько теплоты нужно для превращения воды массой 3 кг при температуре 100°C в пар?

**Дано**:
- m = 3 кг
- L = 2260 кДж/кг = 2 260 000 Дж/кг (вода)
- tₖ = 100°C (температура кипения)

**Найти**: Q

**Решение**:
1. Переведем удельную теплоту парообразования в джоули:
   L = 2 260 000 Дж/кг

2. Подставим в формулу:
   Q = L · m = 2 260 000 Дж/кг · 3 кг = 6 780 000 Дж

3. Переведем в более удобные единицы:
   Q = 6780 кДж

**Ответ**: Q = 6 780 000 Дж = 6780 кДж

---

### Пример 2: Найти массу пара

**Задача**: Какая масса воды превратилась в пар, если для парообразования было затрачено 4520 кДж теплоты?

**Дано**:
- Q = 4520 кДж = 4 520 000 Дж
- L = 2260 кДж/кг = 2 260 000 Дж/кг (вода)

**Найти**: m

**Решение**:
1. Переведем все в джоули:
   Q = 4 520 000 Дж
   L = 2 260 000 Дж/кг

2. Из формулы Q = L · m выразим m:
   m = Q / L

3. Подставим значения:
   m = 4 520 000 Дж / 2 260 000 Дж/кг = 2 кг

**Ответ**: m = 2 кг

---

### Пример 3: Полный процесс (нагревание + парообразование)

**Задача**: Вода массой 2 кг при температуре 15°C нагрета до пара при температуре 100°C. Найдите общее количество теплоты.

**Дано**:
- m = 2 кг
- t₁ = 15°C
- tₖ = 100°C
- cводы = 4200 Дж/(кг·°C)
- L = 2 260 000 Дж/кг

**Найти**: Qобщая

**Решение**:

**Этап 1**: Нагревание воды от 15°C до 100°C
- Δt₁ = 100°C - 15°C = 85°C
- Q₁ = cводы · m · Δt₁ = 4200 Дж/(кг·°C) · 2 кг · 85°C = 714 000 Дж = 714 кДж

**Этап 2**: Парообразование воды при 100°C
- Q₂ = L · m = 2 260 000 Дж/кг · 2 кг = 4 520 000 Дж = 4520 кДж

**Общее количество теплоты**:
- Qобщая = Q₁ + Q₂ = 714 кДж + 4520 кДж = 5234 кДж

**Ответ**: Qобщая = 5234 кДж

---

### Пример 4: Конденсация

**Задача**: Сколько теплоты выделится при конденсации 1,5 кг пара при температуре 100°C?

**Дано**:
- m = 1,5 кг
- L = 2260 кДж/кг = 2 260 000 Дж/кг (удельная теплота конденсации воды)
- tₖ = 100°C (температура конденсации)

**Найти**: Q

**Решение**:
1. При конденсации используется та же формула:
   Q = L · m

2. Подставим значения:
   Q = 2 260 000 Дж/кг · 1,5 кг = 3 390 000 Дж = 3390 кДж

**Ответ**: Q = 3 390 000 Дж = 3390 кДж (выделится при конденсации)

## 15. СВЯЗЬ С ПРЕДЫДУЩИМИ ТЕМАМИ

### Связь с количеством теплоты:

**Нагревание/охлаждение**: Q = c · m · Δt (энергия при изменении температуры)

**Плавление**: Q = λ · m (энергия для перехода из твердого в жидкое)

**Парообразование**: Q = L · m (энергия для перехода из жидкого в газообразное)

**Конденсация**: Q = L · m (энергия при переходе из газообразного в жидкое)

**Горение**: Q = q · m (энергия при химической реакции)

**Все** - это энергия, измеряется в джоулях (Дж)!

### Связь с агрегатными состояниями:

- **Плавление**: твердое → жидкое (Q = λ · m)
- **Кристаллизация**: жидкое → твердое (Q = λ · m)
- **Испарение**: жидкое → пар (при любой температуре, с поверхности)
- **Кипение (парообразование)**: жидкое → пар (при температуре кипения, во всем объеме, Q = L · m)
- **Конденсация**: пар → жидкое (Q = L · m)

## 16. ВАЖНЫЕ ЗАМЕЧАНИЯ

1. **Удельная теплота парообразования = Удельная теплота конденсации** для каждого вещества (Lₚ = Lₖ).

2. **Единицы измерения**:
   - Удельная теплота парообразования: [L] = Дж/кг или кДж/кг
   - Количество теплоты: [Q] = Дж или кДж
   - Масса: [m] = кг

3. **При кипении** температура не меняется, пока не выкипит вся жидкость.

4. **При конденсации** температура не меняется, пока не сконденсируется весь пар.

5. **Полный процесс** (нагревание + парообразование) требует суммы всех теплот:
   Qобщая = Q₁ + Q₂

6. **Вода** имеет самую большую удельную теплоту парообразования среди обычных веществ (2260 кДж/кг).

7. **Кипение отличается от испарения**:
   - Кипение: во всем объеме, при температуре кипения, быстрое
   - Испарение: с поверхности, при любой температуре, медленное
