## ЧТО НУЖНО ВЫУЧИТЬ К СЛЕДУЮЩЕМУ УРОКУ

### Основная формула (обязательно):
**Q = c · m · Δt**

где:
- Q - количество теплоты [Дж]
- c - удельная теплоемкость [Дж/(кг·°C)]
- m - масса [кг]
- Δt - изменение температуры [°C]

### Удельные теплоемкости (выучить наизусть):
- **Вода**: c = 4200 Дж/(кг·°C) - самая большая!
- **Железо**: c = 460 Дж/(кг·°C)
- **Медь**: c = 400 Дж/(кг·°C)
- **Алюминий**: c = 900 Дж/(кг·°C)

### Важные правила:
1. Масса всегда в **килограммах** (кг)
2. При нагревании: Q > 0, Δt > 0
3. При охлаждении: Q < 0, Δt < 0
4. Изменение температуры: Δt = tконечная - tначальная

### Единицы измерения:
- 1 кДж = 1000 Дж
- 1 г = 0,001 кг
- 1 кг = 1000 г
